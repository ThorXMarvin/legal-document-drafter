# Legal Document Drafter Agent вҖ” Concept Note

## Overview
An AI agent that generates common legal documents for Ugandan/East African businesses. Users answer guided questions and get professionally formatted documents вҖ” employment contracts, NDAs, service agreements, tenancy agreements, and more. All templates aligned with Ugandan law.

## Problem
Small businesses and individuals in East Africa pay UGX 200,000-2,000,000 for basic legal documents. Many operate without proper contracts, exposing them to legal risk. Templates found online are based on US/UK law and don't apply locally. Legal services are inaccessible for most SMEs.

## Solution
A configurable legal document agent that:
- Generates common legal documents through guided conversation
- Uses templates aligned with Ugandan/East African law
- Fills in details based on natural language answers
- Outputs professional PDFs
- Supports multiple document types (contracts, agreements, letters)
- Includes legal disclaimers (not a substitute for legal advice)
- Supports English and local language summaries

## Tech Stack
- **Runtime:** Node.js
- **LLM:** Any provider (for guided Q&A and document refinement)
- **PDF:** Puppeteer or PDFKit (professional PDF generation)
- **Templates:** Handlebars/Mustache (legal document templates)
- **Channels:** Baileys (WhatsApp), Telegraf (Telegram), Express (Web)
- **Database:** SQLite (generated documents, user history)

## Configuration
```bash
git clone https://github.com/AiStudioUg/legal-document-drafter.git
cd legal-document-drafter
cp .env.example .env
nano config/settings.yaml
npm install && npm start
```

### config/settings.yaml
```yaml
jurisdiction: "uganda"  # uganda | kenya | tanzania | east-africa

documents:
  employment_contract:
    enabled: true
    includes: ["probation", "termination", "confidentiality", "leave"]
  nda:
    enabled: true
    types: ["mutual", "one-way"]
  service_agreement:
    enabled: true
  tenancy_agreement:
    enabled: true
    includes: ["residential", "commercial"]
  sale_agreement:
    enabled: true
  partnership_deed:
    enabled: true
  demand_letter:
    enabled: true
  company_resolution:
    enabled: true

output:
  format: "pdf"  # pdf | docx | both
  letterhead: true
  watermark: "DRAFT вҖ” Review with legal counsel before signing"

llm:
  provider: "openai"
  model: "gpt-4o"
  api_key: "${OPENAI_API_KEY}"

disclaimer:
  enabled: true
  text: "This document was generated by AI and is for reference purposes only. It is not a substitute for professional legal advice. Please consult a qualified lawyer before signing."
```

## Document Types Supported
1. **Employment Contract** вҖ” aligned with Uganda Employment Act 2006
2. **Non-Disclosure Agreement** вҖ” mutual and one-way
3. **Service Agreement** вҖ” freelancer/contractor engagements
4. **Tenancy Agreement** вҖ” residential and commercial, aligned with Landlord and Tenant Act
5. **Sale Agreement** вҖ” goods and property
6. **Partnership Deed** вҖ” business partnerships
7. **Demand Letter** вҖ” debt collection, breach notices
8. **Company Resolution** вҖ” board and shareholder resolutions
